## Visão geral

As tags no módulo de Reputação Online funcionam de forma similar às tags do módulo de Pesquisas.

<Callout kind="info" collapsed="false">
  O sistema pode levar alguns minutos para aplicar tags em avaliações já recebidas após o salvamento da configuração.
</Callout>

---

## Passo a passo

<Steps>
  <Step title="Acesse o módulo de Reputação Online" title-type="p">
    Na barra lateral da plataforma, clique no ícone de estrela `⭐`.
  </Step>

  <Step title="Crie uma Tag" title-type="p">
    Clique em `Editar tags`. Defina um nome para o tema e adicione as palavras-chave que representam esse assunto.

    **Exemplo:**

    - Tags: `Estrutura`

    - Palavras-chave: `limpeza`, `organização`, `conforto`, `acessibilidade`, `estacionamento`, `manutenção`, `acústica`

    Cada palavra-chave cadastrada será usada para identificar automaticamente esse tema nas avaliações recebidas.

    <Callout kind="tip" collapsed="false">
      Agrupe palavras com significado similar dentro de uma mesma tag. Quanto mais abrangente o vocabulário, maior a cobertura automática nas avaliações.
    </Callout>

    <Image src="https://cdn.amplifique.me/central-de-ajuda/baee1fa08f83c0356bf20dc0cf46a200.png" width="558" height="369" />
  </Step>

  <Step title="Crie um conjunto de tags (opcional)" title-type="p">
    Conjuntos agrupam múltiplas tags sob um mesmo guarda-chuva temático, facilitando análises por categoria maior.

    Clique em `Editar Conjunto de `tags, nomeie o conjunto e adicione as tags desejadas.

    **Exemplo de estrutura:**

    | Conjunto        | tags      | Palavras-chave              |
    | --------------- | --------- | --------------------------- |
    | OB – Onboarding | Estrutura | limpeza, conforto, acústica |
    | OB – Onboarding | Música    | som, instrumento, show      |

    <Callout kind="tip" collapsed="false">
      Use o nome do conjunto para refletir uma área ou etapa da jornada do cliente — ex: `Onboarding`, `Pós-venda`, `Produto`.
    </Callout>
  </Step>
</Steps>

---

## Como funciona na prática

Após salvar as tags, o sistema analisa automaticamente o texto das avaliações e aplica o tema correspondente quando uma palavra-chave é encontrada.

**Exemplo:**

Você criou a tag `Reserva` com as palavras-chave: `horário`, `atendimento`, `café`.

Uma avaliação que contenha: *"Gostei do atendimento no horário marcado, mas o café estava frio."* será automaticamente marcada com a tag **Reserva** — pois contém as palavras `atendimento`, `horário` e `café`.

<Callout kind="info" collapsed="false">
  A aplicação retroativa funciona automaticamente após salvar a configuração. Não é necessário reaplicar manualmente para avaliações antigas.
</Callout>

## Perguntas frequentes

<ExpandableGroup>
  <Expandable title="Qual a diferença entre tag e conjunto de tags?" default-open="true">
    Uma tag representa um tema específico e é composta por palavras-chave. Um **conjunto de Tags** é um agrupamento de múltiplas tags sob uma categoria maior. Por exemplo: o conjunto `Produto` pode conter as tags `Qualidade`, `Embalagem` e `Prazo de entrega`.
  </Expandable>

  <Expandable title="As tags são aplicadas em tempo real nas novas avaliações?" default-open="false">
    Sim. Novas avaliações recebidas após a configuração são tagueadas automaticamente assim que chegam. Para avaliações já recebidas, o sistema processa em alguns minutos após o salvamento da tag.
  </Expandable>

  <Expandable title="Posso usar as tags para filtrar avaliações na tela?" default-open="false">
    Sim. Após as avaliações serem tagueadas, você pode filtrar a aba **Avaliações** por tag para analisar temas específicos e identificar padrões nos feedbacks.
  </Expandable>

  <Expandable title="Uma avaliação pode receber mais de uma tag?" default-open="false">
    Sim. Se o texto da avaliação contiver palavras-chave de tags diferentes, todas elas serão aplicadas simultaneamente.
  </Expandable>
</ExpandableGroup>
