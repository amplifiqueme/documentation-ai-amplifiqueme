## Visão geral

O **importador de respostas** permite trazer respostas de outras fontes — sistemas legados, planilhas históricas ou outras plataformas de pesquisa — para dentro da Amplifique.me. As respostas importadas aparecem nos relatórios junto com as demais respostas da pesquisa.

<Callout kind="alert">
  Respostas importadas **consomem saldo de respostas** da sua conta, da mesma forma que respostas recebidas organicamente.
</Callout>

---

## Como acessar o importador

Existem duas formas de acessar:

- Na tela de **Pesquisas**, clique nos três pontos (`⋯`) ao lado da pesquisa desejada → **Importar respostas**
- Pelo menu lateral → três pontos (`⋯`) → **Importações → Importar respostas**

<Callout kind="tip">
  Recomendamos criar uma **pesquisa separada exclusiva para importação**, para manter o histórico isolado das respostas recentes. Se quiser somar as respostas no futuro, use os **mesmos nomes internos de perguntas** da pesquisa principal — caso contrário, não será possível consolidar as perguntas nos relatórios.
</Callout>

---

## Formatos aceitos por tipo de dado

| Campo | Regras de formato |
|---|---|
| **NPS** | Número inteiro de 0 a 10 |
| **CSAT** | Número inteiro de 1 a 5 |
| **CES** | Número inteiro de 1 a 7 |
| **Única Escolha** | Texto com o valor exato da opção |
| **Múltipla Escolha** | Texto separado por delimitador (escolhido no 2º passo da importação) |
| **Pergunta Aberta** | Texto livre |
| **Caixa de Seleção** | Texto |
| **Like/Dislike** | Aceita: `true`, `1`, `like`, `up` para positivo; `false`, `-1`, `0`, `dislike`, `down` para negativo |
| **Matriz de Opções** | Texto com o valor exato das colunas da matriz |
| **Escala Likert** | Número inteiro conforme a escala configurada na pesquisa |
| **Nome do Contato** | Texto |
| **E-mail do Contato** | Texto com e-mail válido |
| **Telefone do Contato** | Texto com telefone válido (com DDD) |
| **Data de Envio** | `DD/MM/YYYY`, `YYYY-MM-DD` ou ISO 8601 |
| **Data de Resposta** | Mesmo formato da Data de Envio |
| **Campos Customizados** | Texto |

## Perguntas frequentes

<ExpandableGroup>

<Expandable title="Preciso usar a mesma pesquisa para importar ou posso criar uma nova?">
  Você pode usar qualquer pesquisa existente ou criar uma nova. A recomendação é criar uma pesquisa separada para a importação. Se quiser que os dados se somem aos relatórios da pesquisa principal no futuro, use os mesmos nomes internos de perguntas nas duas pesquisas.
</Expandable>

<Expandable title="O que acontece se eu usar nomes internos diferentes dos da pesquisa original?">
  As respostas serão importadas normalmente, mas não será possível cruzar ou somar os dados com as perguntas da outra pesquisa nos relatórios de grupo. Cada pergunta é identificada pelo seu nome interno — se os nomes diferem, são tratadas como perguntas distintas.
</Expandable>

<Expandable title="O importador funciona para qualquer tipo de pergunta?">
  Sim, para todos os tipos suportados na plataforma. Verifique a tabela de formatos aceitos — especialmente para Like/Dislike e Múltipla Escolha, que têm regras específicas de formato.
</Expandable>

<Expandable title="Respostas importadas aparecem nos relatórios junto com as demais?">
  Sim. Após a importação, as respostas ficam visíveis na aba Respostas e são contabilizadas nos painéis e relatórios, incluindo NPS, CSAT e AmpliScore.
</Expandable>

</ExpandableGroup>
