---
title: Cannot Upload File
description: Fix upload issues in Amplifique.me.
---

## Symptoms

- Upload fails or freezes
- Error: "File type not supported" or "Upload error"

## Quick Fix

<Steps>
  <Step title="Check file type">
    Only PDF, DOCX, and PNG files are supported.
  </Step>
  <Step title="File size limit">
    Files must be under 20MB.
  </Step>
  <Step title="Stable connection">
    Ensure you are connected to the internet.
  </Step>
  <Step title="Try another browser">
    Use Chrome or Firefox for best results.
  </Step>
</Steps>
