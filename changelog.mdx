## Version 2025.6 (November and December)

<Update label="2024-12-15" description="v2025.6" tags={["feature", "improvement"]}>

## New Features

- Enhanced AI-powered response analysis with deeper insights into customer sentiment and trends.
- New metrics dashboard showing response rates and engagement over time.

## Improvements

- Updated user permission rules for better team collaboration and access control.
- Faster loading for in-app surveys on web and mobile platforms.

## Bug Fixes

- Resolved issues with bulk contact imports from Excel files.
- Fixed filtering glitches in the contacts list.

</Update>

## Version 2025.5 (September and October)

<Update label="2024-10-10" description="v2025.5" tags={["feature", "bugfix"]}>

## New Features

- Introduced advanced tagging system for contacts and survey responses.
- Added mass survey sending with customizable scheduling.

## Improvements

- Optimized performance for large contact lists (`>10,000` entries).
- Better integration with external analytics tools.

## Bug Fixes

- Corrected AI insight generation errors for non-English responses.
- Fixed permission denial bugs for admin-level users.

</Update>

## Version 2025.4

<Update label="2024-09-01" description="v2025.4" tags={["bugfix", "improvement"]}>

## Improvements

- Streamlined contact creation workflow with auto-suggestions.
- Enhanced dashboard visualizations for response metrics.

## Bug Fixes

- Patched security vulnerabilities in API endpoints.
- Resolved crashes during high-volume survey deployments.

</Update>
